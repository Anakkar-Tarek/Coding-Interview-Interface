# Online Coding Interview Platform

A minimal end-to-end web application for conducting online coding interviews with real-time collaborative code editing.

Built as part of the **AI Dev Tools Zoomcamp**, this project demonstrates how AI-assisted development can be used to design, implement, test, and containerize a full-stack application.

- Create shareable interview session links

- Unit and integration tests
- Fully containerized with Docker
